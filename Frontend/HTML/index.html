Help

Our interactive chatbot helps you commit code by answering questions. Here's how to navigate the process:

  • Proceed: Answer questions as prompted by the chatbot.
  • Go Back: Type PREVIOUS to correct an answer by revisiting the previous question.
  • Exit Anytime: Enter TERMINATE to stop the process immediately, without making any commits.
  • Finalize: Confirm your final submission by entring CONFIRM
  • Support: For assistance or suggestions, contact our support team at CodeMover User Support.
Generate Github Access Token

Go to Github and Verify your email address if it hasn't been verified.

Navigate to your profile photo in the upper-right corner of any GitHub page and click on it, then select "Settings".

In the left sidebar, click "Developer settings".

Under "Personal access tokens" in the left sidebar, click "Fine-grained tokens".

Click "Generate new token".

Fill in the token details such as "Token name", "Expiration", optionally a "Description", and select the "Resource owner".

Select the repositories the token will access under "Repository access" and choose the necessary permissions under "Permissions".

Click "Generate token" to create your new fine-grained personal access token.

Remember to choose the minimal repository access and permissions necessary for your needs.
